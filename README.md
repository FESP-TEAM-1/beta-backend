# beta-backend

## 프로젝트 사용법

- 구동법

  ```
  $ git clone https://github.com/FESP-TEAM-1/beta-backend.git
  $ npm install
  $ npm run dev
  ```

- .env
  ```
  ENDPOINT_URL=
  
  DB_HOST=
  DB_PORT=
  DB_USER=
  DB_PASSWORD=
  DB_SCHEMA=
  
  JWT_SECRET_KEY=
  
  AWS_ACCESS_KEY_ID=
  AWS_SECRET_ACCESS_KEY=
  
  EMAIL_SERVICE=
  EMAIL_ADDRESS=
  EMAIL_PASSWORD=
  ```
