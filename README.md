# beta-backend

## 프로젝트 사용법

- 구동법

  ```
  $ git clone https://github.com/FESP-TEAM-1/beta-backend.git
  $ npm install
  $ npm run dev
  ```

- .env.development / .env.production
  ```
  PORT=
  ENDPOINT_URL=
  
  DB_HOST=
  DB_PORT=
  DB_USER=
  DB_PASSWORD=
  DB_SCHEMA=
  
  JWT_SECRET_KEY=
  COOKIE_DOMAIN
  
  AWS_ACCESS_KEY_ID=
  AWS_SECRET_ACCESS_KEY=
  AWS_REGION=
  AWS_S3_BUCKET_NAME=
  
  EMAIL_SERVICE=
  EMAIL_ADDRESS=
  EMAIL_PASSWORD=
  
  UNIVCERT_API_KEY=
  TOSSPAYMENTS_API_SECRET_KEY=
  
  SESSION_EXPIRE_TIME=
  SESSION_SECRET_KEY=
  ```
